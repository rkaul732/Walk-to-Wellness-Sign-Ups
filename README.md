# Walk to Wellness Team Sign Ups

A public-facing team registration site for Walk to Wellness.

## Run locally

```bash
npm start
```

Then open:

```text
http://localhost:4173
```

The site stores teams, members, registrations, distance entries, and wall messages in `data/walk-to-wellness.json`.

## Deploy publicly

Use [SETUP.md](./SETUP.md) to connect the project to GitHub, Netlify, and Supabase.
